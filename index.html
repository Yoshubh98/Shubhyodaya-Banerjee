Tags and Tools for HTML

  1. <h1> - Heading

    Used for creating top-level headings. There are six levels of headings, from <h1> to <h6>.

  2. <p> - Paragraph

    Used to create paragraphs of text. It is a block-level element.

  3. <a> - Anchor

    Used for creating hyperlinks. It can link to external resources or other pages within the same website.

  4. <img> - Image

    Used to embed images in the web page. Requires the src attribute to specify the image source.

  5. <ul> - Unordered List

    Used to create bulleted lists. Contains <li> elements for each list item.

  6. <ol> - Ordered List

    Used to create numbered lists. Contains <li> elements for each list item.

  7. <table> - Table

    Used to create tabular data. Contains <tr> (table row), <th> (table header), and <td> (table data) elements.

  8. <form> - Form

    Used to create interactive forms. Contains form elements like <input>, <textarea>, and <button>.

  9. <div> - Division

    Used to group and style content. It's a block-level container that can be styled using CSS.

  10. <span> - Span

    Used for inline styling and grouping of content. It's an inline container that can be styled using CSS.

  1. <style> - Style Block

    Used to define inline CSS styles within the HTML document. Can be placed in the head or directly within elements.

  2. <link> - External Stylesheet

    Used to link an external CSS stylesheet to the HTML document. Placed within the <head> section.

  3. <meta name="viewport">

    Used to control how the webpage is displayed on different devices by setting the viewport properties.

  4. <div> - Division

    Used as a container to group and style content using CSS.

  5. <span> - Span

    Used as an inline container to apply CSS styles to specific parts of the text or content.

  6. <class> - Class Selector

    Used to apply a CSS style to one or more HTML elements. Elements with the same class attribute share the same style.

  7. <id> - ID Selector

    Used to uniquely identify an HTML element for styling or scripting purposes.

  8. <link rel="stylesheet">

    Used within the <head> section to link an external CSS file to the HTML document.

  9. <style scoped>

    Used to limit the scope of the embedded CSS styles to the current element and its descendants.

  10. <style media="media_query">

    Used to define styles that are applied only when the specified media query conditions are met.

  1. color - Color Property

    Sets the color of text. Can be specified using color names, hexadecimal, RGB, or HSL values.

  2. font-family - Font Family Property

    Specifies the font to be used for text. Can be set to a specific font or a list of fallback fonts.

  3. font-size - Font Size Property

    Sets the size of the text. Can be specified in various units such as pixels, ems, or percentages.

  4. margin - Margin Property

    Sets the space outside an element. Can be set individually as margin-top, margin-right, margin-bottom, and margin-left.

  5. padding - Padding Property

    Sets the space inside an element. Can be set individually as padding-top, padding-right, padding-bottom, and padding-left.

  6. background-color - Background Color Property

    Sets the background color of an element. Works well with color values similar to the color property.

  7. border - Border Property

    Defines the style, width, and color of an element's border. Can be set individually as border-width, border-style, and border-color.

  8. text-align - Text Alignment Property

    Controls the horizontal alignment of text within an element. Values include left, right, center, and justify.

  9. display - Display Property

    Determines how an element is displayed. Common values include block, inline, and inline-block.

  10. float - Float Property

    Specifies whether an element should float to the left or right within its container. Useful for creating layouts.
